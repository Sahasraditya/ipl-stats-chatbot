# ipl-stats-chatbot
Chatbot to Query IPL statistics (updated till 2022)

This is a project using Python, Streamlit and Langchain which converts Natural Language to SQL and queries a database. This project queries a database containing ball by ball data from the Indian Premier League and can be used to get basic statistics. The databse is updated until IPL 2022. Please note that this project requires an OpenAI api key to function.
